# Algoritmos de jogos simples

Esse projeto trás alguns exemplos de algoritmos de jogos simples para ajudar iniciantes em programação.

## Jogos implementados:

- **Cara ou Coroa**
- ~~Par ou impar~~
- ~~Adivinhe o número~~
- ~~Pedra, papel ou tesoura~~

## Os algoritmos estão disponíveis nas linguagens:

- **PHP**
- ~~Javascript~~
- ~~Python~~
- ~~Java~~
- ~~Ruby~~
- ~~C~~
- ~~C#~~
